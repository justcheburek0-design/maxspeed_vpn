# MaxSpeed VPN — Устранение неполадок

## Частые проблемы

### Приложение не запускается
- Требуется Android 7.0+ (API 24)
- Очистите кэш: Настройки → Приложения → MaxSpeed VPN → Хранилище → Очистить кэш
- Перезагрузите устройство

### Нет диалога VPN-разрешения
- Настройки → Приложения → MaxSpeed VPN → Разрешения — убедитесь что VPN не заблокирован
- На Xiaomi/Huawei отключите оптимизацию батареи
- Android допускает только один VPN одновременно

### Подключено, но нет интернета
- Смените сервер
- Переключите протокол в Настройках
- Отключите IPv6 если ISP имеет проблемы с маршрутизацией

### Частые отключения
- Включите Kill Switch в настройках
- Используйте WireGuard (лучшее NAT-пробитие)
- Отключите режим энергосбережения для приложения

## Логи

Уровни: **INFO**, **WARN**, **ERROR**, **DEBUG**

| Паттерн | Значение | Действие |
|---------|----------|----------|
| `TLS handshake failed` | Несоответствие сертификата | Обновите приложение |
| `AUTH_FAILED` | Неверные данные | Обновите подписку |
| `Connection timed out` | Сервер недоступен | Смените порт/сервер |
| `DNS resolution failed` | Не резолвится хост | Проверьте DNS |
| `TUN/TAP device error` | Ошибка VPN-интерфейса | Перезагрузите устройство |

## Ошибки подписок

- **«Subscription Not Found»**: сверьте Google-аккаунт, Настройки → Аккаунт → Восстановить покупки
- **«Payment Failed»**: обновите способ оплаты в Play Store
- **Подписка активна, но фичи заблокированы**: остановите приложение, очистите данные, войдите заново

## Производительность

- **Высокий расход батареи**: используйте WireGuard, отключите «Always-on VPN»
- **DNS-утечки**: включите защиту от DNS-утечек в настройках, проверьте на dnsleaktest.com
- **Split tunneling**: включите в настройках, добавьте приложения в список исключений

## Обратная связь

- **В приложении**: Настройки → Помощь → Связаться с нами
- Включите экспорт логов при обращении

При обращении укажите: модель устройства, версию Android, версию приложения, логи.
