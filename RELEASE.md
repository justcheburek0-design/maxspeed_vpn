# Release Notes

## 🚀 НОВОЕ
- Флаги стран теперь отображаются в названии сервера (извлекаются из подписки)
- macOS x86_64 поддержка через Rosetta 2 (один билд для обоих архитектур)

## ИСПРАВЛЕНИЯ
- Windows: исправлен путь к билду и PowerShell синтаксис упаковки
- Linux: бинарник загружается напрямую (без tar.gz)
- Web: убран из релизов (не является продуктом для скачивания)
- macOS: использует macos-latest вместо недоступного macos-13

## ⚡ УЛУЧШЕНИЯ
- Все платформы загружаются без архивов (кроме Windows/iOS)
- macOS: два .dmg файла (arm64 + x86_64) в одном релизе
